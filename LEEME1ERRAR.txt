2. COMANDOS DISPONIBLES (Consola ADMIN de PCMASTER)
================================================================================

COMANDO                              DESCRIPCIÓN
-------------------------------------------------------------------------------
perfiles                             Lista todos los perfiles disponibles
asignar <cant> url <URL> duracion <min>   Asigna perfiles a una URL
comentarios_activar url <URL> nivel <bajo/medio/alto>   Activa IA
comentarios_desactivar url <URL>     Desactiva comentarios
detener url <URL>                    Detiene un grupo
estado                               Muestra estadísticas del sistema
salir                                Detiene el servidor

EJEMPLOS:
-------------------------------------------------------------------------------
asignar 3 url https://kick.com/obaysito duracion 22
comentarios_activar url https://kick.com/obaysito nivel alto
comentarios_desactivar url https://kick.com/obaysito
detener url https://kick.com/obaysito
estado

3. FUNCIONALIDADES IMPLEMENTADAS
================================================================================

✅ CONEXIÓN WEBSOCKET
   - Servidor ASYNC en PCMASTER (puerto 5006)
   - Cliente síncrono en PCBOT
   - Comunicación sobre Tailscale

✅ DETECCIÓN DE PERFILES
   - RoxyBrowser: escanea %APPDATA%\RoxyBrowser\browser-cache\
   - Identifica perfiles por windowName (User1_1, User1_2, etc.)
   - Reporta automáticamente al servidor

✅ APERTURA DE PERFILES
   - Usa API de RoxyBrowser (/browser/open)
   - Recibe WebSocket endpoint (CDP)
   - Playwright conecta y navega a la URL

✅ JARVIS (IA GENERADORA DE COMENTARIOS)
   - Modelo: Ollama con llama3.2
   - Prompt maestro personalizable (prompts/maestro.txt)
   - Memoria de contexto (últimos 30 segundos)
   - Genera comentarios únicos por perfil

✅ GESTIÓN DE GRUPOS
   - Asignación de perfiles por URL
   - Activación/desactivación de comentarios
   - Detención de grupos

✅ INYECCIÓN DE COMENTARIOS
   - Playwright conecta por CDP
   - Selectores: textarea[placeholder*="Message"]
   - Simula escritura humana (delay aleatorio)

4. INSTALACIÓN
================================================================================

PCMASTER (PC SERVIDOR):
-------------------------------------------------------------------------------
1. Ejecutar PowerShell como Administrador
2. Copiar y pegar el script de instalación de PCMASTER
3. Esperar a que termine
4. Ejecutar: %USERPROFILE%\Desktop\ROXYMASTER\PCMASTER\INICIAR.bat

PCBOT (CADA PC CLIENTE):
-------------------------------------------------------------------------------
1. Ejecutar PowerShell como Administrador
2. Copiar y pegar el script de instalación de PCBOT
3. Esperar a que termine
4. Ejecutar: %USERPROFILE%\Desktop\ROXYMASTER\PCBOT\INICIAR.bat

REQUISITOS PREVIOS:
-------------------------------------------------------------------------------
- Windows 10/11 (64 bits)
- Python 3.10 instalado
- Tailscale instalado y autenticado
- RoxyBrowser instalado (solo en PCBOT)
- Ollama instalado y corriendo (solo en PCMASTER)

6. CONFIGURACIÓN DE ROXYBROWSER (PCBOT)
================================================================================

PASOS OBLIGATORIOS:
-------------------------------------------------------------------------------
1. Abrir RoxyBrowser
2. Ir a API → API Configuration
3. Activar el switch "API Status" (ON)
4. Verificar que el puerto sea 50000
5. Copiar el API Key
6. Pegar el API Key en %USERPROFILE%\Desktop\ROXYMASTER\PCBOT\config.json
   en la sección "roxy_api" → "token"

IMPORTANTE:
-------------------------------------------------------------------------------
- RoxyBrowser debe estar ABIERTO antes de ejecutar el PCBOT
- La API debe estar ACTIVADA
- NO cerrar RoxyBrowser mientras el PCBOT esté corriendo

7. PERSONALIZACIÓN DE JARVIS (PCMASTER)
================================================================================

ARCHIVO: %USERPROFILE%\Desktop\ROXYMASTER\PCMASTER\prompts\maestro.txt

Puedes editar este archivo para cambiar la personalidad de JARVIS.
Ejemplo de contenido:

[PROMPT MAESTRO: EL COMENTARISTA NATURAL]

Eres un espectador real de streams. Comentas porque te gusta ser parte del chat.
Usas jerga natural, mayúsculas aleatorias y emojis.
Tienes errores ortográficos: "no see", "vamooo", "oyeee"

REGLAS:
- Máximo 60 caracteres
- Nunca digas que eres una IA
- No uses @menciones

EJEMPLOS:
- "no see, que fue eso jaja"
- "VAMOOO asi se hace 🔥"

8. SOLUCIÓN DE PROBLEMAS COMUNES
================================================================================

PROBLEMA: "No se pudo obtener workspace"
SOLUCIÓN: Verificar que RoxyBrowser esté abierto y la API activada

PROBLEMA: "ECONNREFUSED" al conectar CDP
SOLUCIÓN: Reiniciar RoxyBrowser completamente

PROBLEMA: El PCBOT no se conecta al servidor
SOLUCIÓN: Verificar Tailscale (tailscale status). Ambas PCs deben estar online

PROBLEMA: Los comentarios son genéricos ("no see 🔥")
SOLUCIÓN: Verificar que Ollama esté corriendo y el modelo llama3.2 descargado

PROBLEMA: Error "dictionary changed size during iteration"
SOLUCIÓN: Ya corregido en la última versión del servidor

PROBLEMA: El comando "broadcast" no existe
SOLUCIÓN: Será implementado en próxima actualización
