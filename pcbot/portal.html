guía rápida

  1. abre roxybrowser y activa la api en configuración → api status: on (puerto 50000).
  2. copia el api key que aparece.
  3. en la carpeta del pcbot, edita config.json y pega el api key en "roxy_api" → "token".
  4. crea tus perfiles manualmente dentro de roxybrowser (huella digital única, proxy).
  5. inicia los perfiles que quieras usar.
  6. ejecuta iniciar_pcbot.bat para conectar con el servidor.
