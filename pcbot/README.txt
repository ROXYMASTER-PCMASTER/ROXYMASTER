PCBot - Carpeta funcional
==========================
Esta carpeta contiene solo los archivos necesarios para el agente PCBot.

Estructura:
- scripts/         : código fuente del agente (detección, heartbeat, WS client, etc.)
- data/            : configuración (roxy_config.json, roxy_profiles.json, secrets/)
- .gitignore       : archivos ignorados por Git

Cualquier archivo de respaldo, log o prueba ha sido movido a:
C:\Users\CYBER\Desktop\backups_pcbot/pcbot_historico_...

Si algún script no funciona, revisa esa carpeta de respaldo para recuperar lo necesario.
